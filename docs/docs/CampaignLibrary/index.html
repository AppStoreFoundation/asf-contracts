convertCountryIndexToBytes bf303d14

Converts country index list into 3 uints

Expects a list of country indexes such that the 2 digit country code is converted to an index. Countries are expected to be indexed so a "AA" country code is mapped to index 0 and "ZZ" country is mapped to index 675.

inputs0uint256[]countries

List of country indexes

outputs0uint256countries1

First third of the byte array converted in a 256 bytes uint

outputs1uint256countries2

Second third of the byte array converted in a 256 bytes uint

outputs2uint256countries3

Third third of the byte array converted in a 256 bytes uint
